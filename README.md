# ~~Обкуренный~~ Продвинутый ЕМИАС v4.20

Медицинская информационная система для управления данными пациентов с современным интерфейсом и аналитическими возможностями.

## Описание
  
Приложение "Обкуренный ЕМИАС" представляет собой систему управления медицинскими данными пациентов с расчетом индекса массы тела (ИМТ) и визуализацией статистики. Программа сочетает в себе функциональность с нестандартным дизайном.

## Особенности
  
	Управление пациентами - добавление, редактирование, удаление записей

	Расчет ИМТ - автоматический расчет индекса массы тела

	Визуализация статистики - графики распределения по полу, возрасту и ИМТ

	Современный интерфейс - адаптивный дизайн с фоновым изображением

	Сохранение данных - автоматическое сохранение в JSON файл

	Тестовые данные - генерация реалистичных данных пациентов

	Цветовая схема - гармонирующая с фоном палитра цветов

## Установка и запуск

	Требования
	Python 3.6 или выше

## Установленные зависимости из requirements.txt

Установка зависимостей и запуск приложения
bash
```
pip install -r requirements.txt
python EMIAS_version_4.20.py
```

или код целиком для Linux:
```
git clone https://github.com/biopolly/EMIAS.git
# Если нет необходимых модулей
pip install matplotlib
pip install Pillow    # для картинок
pip install Faker
cd EMIAS
python EMIAS_version_4.20.py
```

## Основные функции

	Добавление пациента - ввод ФИО, возраста, пола, роста и веса

	Редактирование данных - изменение информации о существующем пациенте

	Удаление пациентов - удаление выбранных записей

	Просмотр статистики - визуализация данных через интерактивные графики

## Статистические графики

	Распределение пациентов по полу

	Распределение пациентов по возрасту

	Распределение ИМТ с учетом пола

	Зависимость ИМТ от возраста

## Структура данных
Данные сохраняются в файл patients.json в формате:

json
```
{
  "full_name": "Иванов Иван Иванович",
  "age": 35,
  "gender": "М",
  "height": 180,
  "weight": 75
}
```
## Интерфейс
Адаптивный дизайн - корректное отображение на разных разрешениях

Фоновое изображение - тематический медицинский фон

Цветовая схема - синяя палитра с контрастными элементами

Удобная навигация - интуитивно понятное расположение элементов

## Технические особенности

Кросс-платформенность (Windows, Linux, macOS)

Автоматическое масштабирование интерфейса

Обработка ошибок ввода данных

Поддержка Unicode (русские символы)

## Решение проблем
Если возникают проблемы с отображением графиков:

bash
`pip install --upgrade matplotlib`

При проблемах с изображениями:

bash
`pip install --upgrade Pillow`
